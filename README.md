# Brothers Beyond Addiction Website

A responsive static website for Brothers Beyond Addiction, a support organization focused on helping young men with gambling and nicotine addiction.

## Files

- `index.html` contains the page content and structure.
- `styles.css` contains the full responsive visual design.
- `script.js` handles mobile navigation and reveal animations.
- `assets/brothers-beyond-addiction-logo.png` contains the company logo.

## How to View

Open `index.html` in a browser.

## Customize

- The contact email is currently set to `brotherbeyondaddiction@gmail.com`.
- Update section copy, programs, and organization details as needed.
- Add real crisis/support resources if the organization has preferred local partners.
